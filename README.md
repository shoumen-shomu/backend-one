# User Management API

A simple REST API built with Express.js and MongoDB for user management.

## Features

- Create User
- Get All Users
- Update User
- Delete User

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

## API Endpoints

| Method | Endpoint | Description |
|----------|----------|-------------|
| POST | /registration | Create a new user |
| GET | /allusers | Get all users |
| POST | /user/:id | Update a user |
| DELETE | /user/:id | Delete a user |



Server runs on:

```bash
http://localhost:5000
```
